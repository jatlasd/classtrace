# ClassTrace asset kit

This folder contains the generated landing-page mockup and cropped reference assets you can use while building the real ClassTrace landing page.

## Files
- `classtrace-full-mockup.png` — full generated landing page reference.
- `01-header.png` through `07-footer.png` — cropped section references.
- `asset-raw-note-card.png` — hero raw teacher note card reference.
- `asset-reviewed-evidence-card.png` — hero structured evidence card reference.
- `asset-stacy-timeline-card.png` — timeline artifact reference.
- `asset-final-cta.png` — final CTA reference.
- `design-tokens.json` — approximate palette, radii, shadows, and typography suggestions.
- `landing-page-copy.md` — all major landing-page copy in editable text.

## Implementation note
The PNG crops are visual references, not clean SVG/vector components. For production, rebuild the UI in HTML/CSS using the copy and design tokens, then treat these images as direction boards.

Original image size: 846 × 1860px.
